# 🏪 Loja Clandestina CLI

Uma aplicação de terminal (CLI) interativa feita em JavaScript (Node.js) que simula um sistema de compras de uma loja fictícia. O programa possui verificação de idade e exibe artes em ASCII incríveis ao finalizar uma compra!

---

## ✨ Funcionalidades

- **Validação de Maioridade:** Apenas usuários com 18 anos ou mais podem acessar o catálogo.
- **Menu Interativo:** Escolha entre diferentes produtos usando o terminal.
- **Controle de Estoque Simples:** Valida se a quantidade solicitada está disponível e exibe o estoque restante.
- **Arte ASCII:** Exibição de gráficos em texto para ilustrar o produto comprado.

---

## 🛠️ Tecnologias Utilizadas

- **JavaScript (ES6+)**
- **Node.js**
- **Biblioteca [prompt-sync](https://www.npmjs.com/package/prompt-sync)** (para capturar as entradas do usuário no terminal)
